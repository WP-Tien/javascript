null được sử dụng để diễn tả sự vắng mặt có chủ ý của một giá trị đối tượng và là một giá trị nguyên thủy ( primitive value ). Không giống  Undefined , nó không phải là một thuộc tính của đối tượng toàn cục.

    Nó bằng undefined nhưng nó không giống hay đồng nhất ( khác kiểu ) với  undefined . Xem ví dụ để hiểu thêm.

    console.log( null == undefined ); // true
    console.log( null === undefined ); // false 

    Cẩn thận: typeof null là object
    console.log( typeof null ); // objects

    Để kiểm tra đúng nếu một giá trị là null, so sánh nó với biểu thức bằng chính xác ( === )

    var a = null;
    a === null; // true
