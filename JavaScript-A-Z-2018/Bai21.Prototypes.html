Constructor function 
        - Being used with new in order to create new objects 
        - function name is a Noun with first character uppercased 

        function Mouse() {}
        var mickey = new Mouse();

        Normal function 
        - Being called to perform some specifuc task 
        - Function name is a verb following camelCase convention

        function sayHello() {
            console.log('Hello');
        }
        sayHello();
