Nếu bạn có sẵn một tập tin hình ảnh chừa văn bản mong muốn và muốn nó đặt trên một server, bạn có thêm URL of hình ảnh và sau đó thêm một hình ảnh đến thư viên như sau:

    var img = new Image();
    img.src =  'https://i.ytimg.com/vi/zecueq-mo4M/maxresdefault.jpg';
    document.body.appendChild( img );
