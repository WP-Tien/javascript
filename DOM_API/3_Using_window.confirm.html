Phương thức window.confirm() hiển thị một 

    result = window.confirm(message);

    message là chuỗi không bắt buộc được hiển thị trong hộp thoại và kết quả là một giá trị boolean chỉ ra liệu OK hoặc Cancel được chọn (true nghĩa là OK).

    var deleteConfirm = window.confirm("Are you sure you want to delete this ?");

    var deleteForm = window.confirm("Are you sure want to delete this ?");
