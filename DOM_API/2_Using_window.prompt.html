Một cách đơn giản để lấy một người dùng input form bằng cách sử dụng phương thức prompt();
    
    Cú pháp: 
    prompt(text, [default]);
    -text: Văn bản hiển thị trong hộp nhắc nhở.
    -default: Một giá trị mặc định cho trường input ( không bắt buộc ).

    Ví dụ:
    var age = prompt("How old are you?");
    console.log(age); // In giá trị được thêm vào bởi user.
