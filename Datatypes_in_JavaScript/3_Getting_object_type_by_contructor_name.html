Trong thực tế, bạn có thể cần thu hẹp nó thành loại 'đối tượng' thực sự là gì và một cách để làm điều đó là sử dụng
    Tên hàm tạo đối tượng để có được hương vị của đối tượng thực sự là: Object.prototype.toString.call (yourObject)

    1.String

    Object.prototype.toString.call("String");

    2. Number

    Object.prototype.toString.call(42);

    3. Bool

    Object.prototype.toString.call( true );

    4. Object

    Object.prototype.toString.call(Object()) or
    Object.prototype.toString.call({});

    5. Function

    Object.prototype.toString.call( function(){} );

    6. Date

    Object.prototype.toString.call( new Date(2015,10,21) );

    7. Regex

    Object.prototype.toString.call( new RegExp() ) or
    Object.prototype.toString.call(/foo/);

    8. Array

    Object.prototype.toString.call([]);

    9. Null

    Object.prototype.toString.call(null);

    10. Undefined

    Object.prototype.toString.call( undefined );

    11. Error

    Object.prototype.toString.call( Error() );
